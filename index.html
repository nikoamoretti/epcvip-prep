Ping-Post Deep Dive

Mechanics, buyer config, pipelines, publisher signals

How Ping-Post Works

Only explain if he tests you.

  1. Consumer fills form on landing page (EPCVIP-owned or affiliate)
  2. Consent captured via TrustedForm
  3. PII stripped → ping created with attributes only (state, ZIP, credit range, loan amount, income, employment)
  4. Ping broadcast to all matching buyers simultaneously
  5. Buyers bid within milliseconds
  6. Platform decides exclusive vs shared based on total revenue max
  7. Winner(s) receive the post: full PII + consent cert
  8. If rejected → waterfall to next bidder (recovery revenue)

Key metrics: fill rate, acceptance rate, return rate, EPC
